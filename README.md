# Open-Source AI Assistant

An open-source AI assistant focused on education and developer productivity.

## Phase 1 Features
- Conversational AI (Q&A, reasoning)
- Code assistance (C++, Python, Web)
- Math and logic problem solving
- PDF document understanding (RAG)
- English + Hindi support

## Tech Stack
- FastAPI (Backend)
- Open-source LLMs (Qwen / DeepSeek)
- FAISS Vector Search
- React / Next.js Frontend
